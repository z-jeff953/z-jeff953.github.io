第1章 计算机网络和因特网

主机或端系统通过_________接入因特网。

  • A.ISP
  • B.调制解调器
  • C.交换机
  • D.路由器

主机一般被划分为两类：客户和________。

  • A.数据中心
  • B.服务器
  • C.端系统
  • D.大型主机

接入网(access network)是指将端系统连接到__________的物理链路。

  • A.路由器
  • B.边缘路由器
  • C.服务器
  • D.因特网

以下_________属于家庭住宅常使用的网络接入方式。

  • A.3G
  • B.以太网
  • C.Wi-Fi
  • D.ADSL

用户通常以_________速率接入以太网交换机。

  • A.100Mbps
  • B.10Mbps
  • C.10Gbps
  • D.1Gbps

假定在发送主机和接收主机间只有一台分组交换机。发送主机和交换机间以及交换机和接收主机间的传输速率分别是R1和R2。假设该交换机使用存储转发分组交换方式，发送一个长度为L的分组的端到端总时延是________。（忽略排队时延、传播时延和处理时延)

  • A.2L / R2
  • B.2L / R1
  • C.L / R1 + L / R2
  • D.L / (R1 + R2)

以下哪种时延受到网络拥塞情况的影响。

  • A.传播时延
  • B.处理时延
  • C.排队时延
  • D.传输时延

一个长度为1000字节的分组经过距离为2900km的2Mbps的以太网链路，其中传播速率为2.9 * 10^8 m/s，其中 传播时延为多少？

  • A.1 s
  • B.1 ms
  • C.0.1 s
  • D.10 ms

9

主机A向主机B发送一个大小为6MB的文件，从主机A到主机B的路径上有3段链路， 其中速率分别是1Mbps, 800kbps, 2Mbps，该文件从主机A传输到主机B大概需要多久?

  • A.48 s
  • B.7.5 s
  • C.60 s
  • D.24 s

因特网协议栈包含哪五层协议？

  • A.应用层、运输层、会话层、网络层、物理层
  • B.应用层、表示层、网络层、链路层、物理层
  • C.应用层、表示层、运输层、链路层、网络层
  • D.应用层、运输层、网络层、链路层、物理层

在网络协议栈中，哪一层负责端到端之间的数据报文的发送与接收？

  • A.运输层
  • B.数据链路层
  • C.网络层
  • D.应用层

SMTP、FTP、TCP、IP协议分别属于哪一层协议？

  • A.应用层、传输层、传输层、网络层

  • B.传输层、应用层、传输层、网络层

  • C.应用层、应用层、传输层、网络层

  • D.应用层、传输层、网络层、网络层

分层协议中，发送发与接收方的数据传递方向是？

  • A.发送方从下到上，接收方从下到上

  • B.发送发从上到下，接收方从下到上

  • C.发送方从下到上、接收方从上到下

  • D.发送发从上到下，接收方从上到下

以下说法中正确的是_________。

  • A.TCP是面向连接的，UDP是面向连接的

  • B.电路交换网络分为虚电路网络和数据报网络

  • C.分组交换网络分为时分复用和频分复用两种

  • D.虚电路网络一定是面向连接的

路径选择功能是在OSI模型的_________上完成的。

  • A.数据链路层

  • B.物理层

  • C.运输层

  • D.网络层
