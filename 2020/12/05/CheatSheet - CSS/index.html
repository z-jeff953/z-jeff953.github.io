CSS 速查表

1. 通过选择器选择元素

MDN

  1. 元素选择器 使用标签名选择元素，可以应用在所有的标签元素

  2. 类选择器 每一个元素可以有多个类，不同种类的标签可以有同一个类名，通过 .类名 来定义属性，标签内使用class = “类名” 来使用类。

  3. ID选择器 一个ID只能使用一次，每个标签只能由一个类名，类名和标签存在一一对应的关系，使用 #ID名 来定义ID，标签内使用 id = “ID名” 来使用id

  4. 属性选择器 可以选择具有某个属性或者某个属性等于特定值的所有标签，使用**[属性名或者属性名=特定值]**来进行选择

  5. 后代选择器 可以选择 某个标签的所有特定标签的后代，使用 父元素（空格）后代 进行选择

  6. 子元素选择器 可以选择父元素的第一个特定元素后代， 使用 父元素 > 子元素 选择

  7. 临近兄弟选择器 可以选择具有相同父元素的元素的下一个特定标签兄弟，使用 子元素1+ 子元素2 选择子元素2，值得一提的是，兄弟选择器对于同级的三个同名元素，会对第二三个元素都产生作用

  8. 同级兄弟选择器

    兄弟选择符，位置无须紧邻，只须同层级，A~B 选择A元素之后所有同层级B元素

  9. 伪类选择器 可以对于元素的不同状态进行选择，比如标签的访问，鼠标滑过等时候进行特殊的格式设置，使用 元素名 : 伪类 进行使用

  10. 伪元素选择器 可以选择伪元素，比如标签的后面，前面等，他会在所有选择的元素对应位置插入相应的伪元素，使用 元素名: 伪元素名使用

  11. 组选择器

    选择一组元素, 之间使用 逗号, 分隔

3. 设置元素的位置和与其他元素的间距

3.1 触发BFC常用条件(块格式化上下文)

  • 浮动元素（元素的 float 不是 none）

  • 绝对定位元素（元素的 position 为 absolute 或 fixed）

  • 行内块元素（元素的 display 为 inline-block）

  • 表格单元格（元素的 display为 table-cell，HTML表格单元格默认为该值）

  • 表格标题（元素的display为 table-caption，HTML表格标题默认为该值）

  • 匿名表格单元格元素（元素的display为 table、``table-row、 table-row-group、``table-header-group、``table-footer-group（分别是HTML table、row、tbody、thead、tfoot 的默认属性）或 inline-table）

  • overflow 值不为 visible 的块元素

  • 弹性元素（display为 flex 或 inline-flex元素的直接子元素）

  • 网格元素（display 为 grid 或 inline-grid 元素的直接子元素）
