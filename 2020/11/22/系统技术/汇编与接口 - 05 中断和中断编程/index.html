汇编与接口 - 05 中断和中断编程.md

8086 机的中断和 8259A 的编程

1. 8086/8088 的中断过程

STI,CLI 是指令

通过编程进入软件中断用INT,返回用INTR

INT 指令执行时, 发送中断请求,关中断, 保护硬件现场(FLAGS 入栈),保护断点(CS,IP 入栈),获得终端服务子程序

然后执行中断服务子程序, 中断服务子程序中保护软件现场(中断程序中需要改变的寄存器值入栈), 然后开中断[允许多级中断], 执行中断服务, 关中断, 恢复软件现场, 然后 IRET 指令

IRET 指令执行时, 恢复断点(CS,IP 出栈), 恢复硬件现场(FLAGS 出栈), 开中断, 中断返回

几步开关中断为的是保护堆栈内容的逻辑不被其他中断打断

2. 8086/8088 微机的中断有哪些(了解)

2.1 8086/8088 内中断类型号

中断类型 功能
类型 0 除数为 0 中断例行程序
类型 1 单步
类型 2 非屏蔽中断， NMI
类型 3 设置断点
类型 4 溢出处理中断，INTO 指令
类型 10 显示设备中断
类型 20 程序结束中断
类型 21 DOS 系统功能调用功能程序

2.2 8086/8088 外中断处理

非屏蔽中断

  • NMI 引脚上出现上升沿触发
  • 不受标志位 IF 的限制，即不可以屏蔽。
  • 类型号 ：2

可屏蔽中断

  • INTR 引脚输入，高电平有效
  • 受标志位 IF 的限制
  • 类型号：08H~0FH 70H~77H

*3. 8086/8088 中断向量和中断编程

3.1 8086/8088 中断向量的特点

  1. 8086/8088中断向量表位于内存最低 1KB, 就是从 00000H 开始的 1KB 内存(00000H~003FFH)

  2. 一共有 256 个中断向量

  3. 每个入口占用 4 Bytes，低字为段内偏移，高字为段基址；

3.3 初始化 8259A

初始化命令字要完成：

  1. 设定中断请求信号的触发方式，是高电平有效还是上升沿有效。
  2. 工作于单片方式还是工作于级连方式。
  3. 设置中断类型码。
  4. 若为级连工作方式，则规定主 8259A 中每个 IR 端是否带从 8259A，从 8259A 则要规定由主 8259A 的哪个 IR 端引入。

4. DOS 操作

调用DOS操作需要把对应的操作号送到AH,然后实用INT 21H调用DOS功能

AH 功能 调用参数 返回参数
00 程序终止（同 INT 21H） CS=程序段前缀 PSP
01 键盘输入并回车 AL=输入字符
02 显示输出 DL=输出字符
07 键盘输入（无回显） AL=输入字符
09 显示字符串 DS：DX=串地址 字符串以‘$’结尾
0A 键盘输入字符串到缓冲区 DS：DX=缓冲区首址
（DS：DX）=缓冲区最大字符数		 （DS：DX+1）=实际输入的字符数
DS:DX+2 字符串首地址
