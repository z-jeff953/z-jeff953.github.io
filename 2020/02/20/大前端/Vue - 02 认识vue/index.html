组件是什么? 可以复用的独立ui模块

vue组件是vue的实例 , 可以通过new Vue来创建一个实例通过不同的构造方法实现不同的效果 ,即:

Vue组件 == Vue实例 == newVue(options)​

我们的核心工作就是配置options

三大核心概念是 :

  1. 属性
  2. 事件
  3. 插槽
