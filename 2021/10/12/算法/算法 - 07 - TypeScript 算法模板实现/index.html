2.1.1 算法描述

深度优先搜索是一种搜索算法 , 核心理念是在遇到分支 ( 多选一 ) 的时候一个一个选择 , 直到没有选择的余地为止 .

时间复杂度

计算方法 : $O (方案个数 * 构造每个方案的时间复杂度)$

  1. 树的遍历 $O(n)$ , $n$ 为树的子叶节点数目
  2. 排列 : $O(n! * n)$ , $n$ 为数组的元素个数
  3. 组合 : $O(2^n * n)$ , $n$ 为数组的元素个数

2.1.2 使用场景

可以使用的场景

  1. 找到满足某个条件的所有方案 ( 99 % )
  2. 二叉树 ( Binary Tree ) 问题 ( 90 % )
  3. 组合 $C^n_m$ 问题 ( 95 % )
  4. 排列 $A^n_m$ 问题 ( 95 % )

不能使用的场景

不能用的时候就用BFS

  1. 连通块问题 ( 用BFS , 不然会爆栈 )
  2. 拓扑排序 ( 用BFS , 不然会爆栈 )
  3. 一切能用BFS 解决的问题

快排基本够用，归并可以用来寻找排好序的数组的中间值，堆排序可以用来解决 Top K

  • 时间复杂度：
    • 快速排序(期望复杂度) ： O(nlogn)
    • 归并排序(最坏复杂度) ： O(nlogn)
  • 空间复杂度：
    • 快速排序 ： O(1)
    • 归并排序 ： O(n)
