React - RenderProps-组件的render项是什么意思

Antd的Table组件中有一个render项目, 可以通过传入一个回调函数处理对应的value值和record在不同的列生成相应的组件.

ReactRouter中也有这样的用法, 用来验证一些登录信息,进行异步操作之类的

那么这玩意是怎么实现的呢, 自己实现一个类似的React组件效果

1. RenderProps

这样,我们就可以通过属性将内容传入组件了, 吧props单独拆分为一个对象, 可以得到更像Table组件的render方法

我们可以看出实现RenderProps的方式是通过一个特殊的回调函数props来把子组件的渲染委托给父组件
