Redux建立了全局数据的使用, Redux-thunk,Redux-saga解决了异步数据的获取问题, Dva.js封装了Redux和Redux-saga,实现了api的减少和收敛

Redux的基本思想是使用单独的空间来存储数据, 这段数据需要存储的地方, 需要改变数据的方法, 在Redux中有一下几个核心概念:

1.2.1 redux核心5概念

  1. store对象 存放数据和数据操作的仓库,是使用数据需要引入的对象
  2. state变量 存放的数据,通过reducer函数定义初始值和修改访问state的方法
  3. dispatch方法 是View层中调用更改store中state内容的操作
  4. reducer函数 定义state的初始值, dispatch的方式
  5. action对象/数据 dispatch的参数就是action
